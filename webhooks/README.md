# Peeps Builder Webooks

Peeps Builder is designed such that it doesn't require a backend, however certain actions benefit from a backend being available.

## Functions

### Relay

Relay is a serverless function that supports relaying data into Notion.
Using a serverless function allows us to securely hold the Notion API token.

## Usage

### Dev

```sh
pnpm start
```

### Deploy

```sh
pnpm run deploy
```
